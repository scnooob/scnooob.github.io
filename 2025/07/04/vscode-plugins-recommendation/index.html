VS Code必备插件推荐

Visual Studio Code作为当前最受欢迎的代码编辑器之一，其强大的扩展生态系统是它的一大亮点。今天我来分享一些我认为非常实用的VS Code插件。

通用插件

1. Chinese (Simplified) Language Pack

插件ID: MS-CEINTL.vscode-language-pack-zh-hans

中文语言包，让VS Code界面显示中文，对于中文用户来说必不可少。

2. Bracket Pair Colorizer 2

插件ID: CoenraadS.bracket-pair-colorizer-2

为括号添加不同颜色，帮助快速识别匹配的括号对，提高代码可读性。

3. Indent Rainbow

插件ID: oderwat.indent-rainbow

为缩进添加彩虹色，让代码层次结构更清晰。

4. GitLens

插件ID: eamodio.gitlens

增强Git功能，可以看到每行代码的提交信息、作者、时间等，非常有用。

5. Live Server

插件ID: ritwickdey.LiveServer

快速启动本地服务器，实时预览HTML文件，前端开发必备。

代码编辑

6. Auto Rename Tag

插件ID: formulahendry.auto-rename-tag

自动重命名HTML/XML标签，修改开始标签时自动修改结束标签。

7. Emmet

内置插件，支持HTML和CSS的快速编写。

8. IntelliSense for CSS class names

插件ID: Zignd.html-css-class-completion

CSS类名智能提示，在HTML中写class时自动提示已定义的CSS类。

主题和外观

9. One Dark Pro

插件ID: zhuangtongfa.Material-theme

经典的暗色主题，界面美观，护眼。

10. Material Icon Theme

插件ID: PKief.material-icon-theme

Material Design风格的文件图标主题。

11. Dracula Official

插件ID: dracula-theme.theme-dracula

另一个流行的暗色主题。

前端开发

12. ES7+ React/Redux/React-Native snippets

插件ID: dsznajder.es7-react-js-snippets

React开发的代码片段，提高开发效率。

13. Prettier - Code formatter

插件ID: esbenp.prettier-vscode

代码格式化工具，支持多种语言。

14. ESLint

插件ID: dbaeumer.vscode-eslint

JavaScript代码质量检查工具。

15. TypeScript Importer

插件ID: pmneo.tsimporter

TypeScript自动导入工具。

后端开发

16. Python

插件ID: ms-python.python

Python开发必备，提供语法高亮、调试、智能提示等功能。

17. Go

插件ID: golang.Go

Go语言开发支持。

18. REST Client

插件ID: humao.rest-client

在VS Code中直接测试REST API。

数据库

19. SQLTools

插件ID: mtxr.sqltools

数据库连接和查询工具。

20. MongoDB for VS Code

插件ID: mongodb.mongodb-vscode

MongoDB数据库支持。

文档和写作

21. Markdown All in One

插件ID: yzhang.markdown-all-in-one

Markdown编写增强工具，支持目录生成、快捷键等。

22. markdownlint

插件ID: DavidAnson.vscode-markdownlint

Markdown语法检查。

23. Paste Image

插件ID: mushan.vscode-paste-image

快速粘贴图片到Markdown文档。

效率工具

24. Todo Tree

插件ID: Gruntfuggly.todo-tree

在代码中查找并高亮显示TODO注释。

25. Bookmarks

插件ID: alefragnani.Bookmarks

代码书签功能，快速导航到重要代码位置。

26. Path Intellisense

插件ID: christian-kohler.path-intellisense

文件路径智能提示。

27. Auto Close Tag

插件ID: formulahendry.auto-close-tag

自动闭合HTML标签。

配置建议

推荐的用户设置

推荐的快捷键

  • Ctrl+Shift+P: 命令面板
  • Ctrl+P: 快速打开文件
  • Ctrl+Shift+E: 文件资源管理器
  • Ctrl+Shift+F: 全局搜索
  • Ctrl+Shift+G: 源代码管理
  • Ctrl+Shift+X: 扩展
  • Ctrl+Shift+D: 调试
  • Ctrl+J: 切换面板
  • Ctrl+B: 切换侧栏

总结

这些插件可以大大提高你的开发效率。当然，不需要一次性安装所有插件，根据自己的需求选择合适的插件。记住，插件虽好，但不要贪多，太多插件可能会影响VS Code的性能。

建议：

  1. 根据你的主要开发语言选择相应插件
  2. 先安装基础插件，再根据需要添加
  3. 定期清理不使用的插件
  4. 关注插件的更新和新功能

希望这些插件推荐能帮助你提高开发效率！如果你有其他好用的插件推荐，欢迎在评论区分享。
